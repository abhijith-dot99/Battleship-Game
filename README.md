# Battleship Game

## Introduction

This is a simple command-line Battleship game implemented in Java. The game simulates the classic board game where two players take turns to guess the location of each other's ships on a grid. The player who sinks all the opponent's ships first wins.

## How to Play

1. Clone the repository or download the source code to your local machine.

2. Compile the Java source code to create the executable class files. You can do this using the `javac` command:

   ```bash
   javac BattleshipGame.java
